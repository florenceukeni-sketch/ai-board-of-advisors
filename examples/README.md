# Example sessions

Real, unedited board sessions exported straight from the app — so you can see the debate mechanic without running anything yourself.

These are the actual outputs, not mockups. Notice what the advisors do: they don't answer in parallel. Each one reads what the advisors before them said and engages with it by name. The Critic reframes the question entirely. Alex sharpens the Critic's point. Mia lands the synthesis. That friction is the whole reason the board exists, and it's the thing a single AI answer can't reproduce.

## Sessions

- **[Should I quit my stable job to go freelance?](board-session-freelance-decision.md)** — Watch the board split, then converge. Marcus runs the cash-flow math, Lena argues the strategic timing, Jared reframes it around AI-augmented capacity, the Critic asks whether the question is even a business one, Alex distinguishes "escaping a job" from "outgrowing a role," and Mia points out the answer was already sitting in an existing client relationship.

Each session is a timestamped markdown file, exactly as the app's export button produces it.
